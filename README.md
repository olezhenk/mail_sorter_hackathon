Сортировщик писем
Программа автоматически читает письма из ZIP-архива и расскладывает их по папкам в зависимости от содержания письма

Как это работает
1.Reader открывает ZIP-архив и пересчитывает все файлы внутри
2.Для каждого .txt-файла LetterExtractor преобразовывает его в объект Letter
3.main_classifier определяет письмо в категорию по набору правил и определению параметра
4.emailsort переносит файл в соответствующий адрес внутри sorted_emails/
5.В конце выводится статистика по категориям

Критерии сортировки
cрочные письма - urgent
письма свыше прям от бога - from_boss
безопасность - security
на подтверждение - need_confirmation
на согласование - need_approval
встречи и созвоны - meeting
финансы - finance
вакансии - vacancy
от роботов - auto_reply
спам - spam

Инструкция по запуску программы :
1) Закинуть архив с письмами "inbox.zip" в корневую папку проекта(важно, чтобы там лежал файл main.py!)
2) Открыть терминал в папке проекта , дать скрипту права для запуска:
 "```bash
    chmod +x run.sh"
3) Запустить сам скрипт :
 "./run.sh"
